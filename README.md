### 合作心得

1. 上傳前記得先同步，本地端解衝突是最容易的
2. 後上傳者要解先上傳者的衝突
3. 後工作者要看先工作者上傳的資源，避免重複上傳或編寫類似內容
4. 讓大家工作可以獨自進行，避免等待而浪費時間
